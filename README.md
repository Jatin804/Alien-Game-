# Alien Game
<h3>A simple 2D old version shooter game, using pygames lib</h3>

<h4>How to run in your device</h4>
<p>Step 1 : Install any IDE, eg : PyCharm</p>
<p>Step 2 : Install python on you device from : <a href="https://www.python.org/downloads/"></a> </p>
<p>Step 3 : Install library : command : <i>pip install pygame </i></p>
<p>Step 4 : Restart your device </p>

<h4>How to run the game</h4>
<p>Step 1 : Open the project in PyCharm</p>
<p>Step 2 : Run the file : <i>main.py</i></p>

<h4>How to play the game</h4>
<p>Use arrow keys to move the spaceship and space key to shoot the bullet</p>
<p>Try to kill the aliens and score the points</p>
<p>Enjoy the game</p>
